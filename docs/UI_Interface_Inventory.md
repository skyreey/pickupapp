```
_root (Stack)
├── onboarding     — 首次启动引导（3页横滑）
├── (tabs)         — 底部 Tab 导航
│   ├── index      — 首页（包裹列表）  [📦 我的包裹]
│   ├── manual     — 手动录入           [✏️ 手动录入]
│   └── settings   — 设置               [⚙️ 设置]
├── detail/[id]    — 包裹详情
├── pro/activate   — 升级 Pro
├── legal/privacy-policy  — 隐私政策
└── legal/user-agreement  — 用户协议
```
