# 取件通 产品路线图 (Roadmap)

## 1. 概述
取件通v1.0已实现核心扫描+管理功能(92%完成度)。v2.0重点在代取体验和用户增长。

## 2. 版本规划

### v2.0 - “代取互联” (当前进行中)
| 功能 | 优先级 | 状态 |
|------|------|------|
| 微信代取监控+弹窗 | P0 | 已实现 |
| 代取包裹过滤视图 | P0 | 已实现 |
| 分享文本格式标准化 | P0 | 待实现 |
| 设置页全部可折叠 | P1 | 待实现 |
| 深色模式 UI 优化 | P1 | 待实现 |
| 代取状态回传(取完通知推送方) | P1 | 待实现 |

### v2.1 - “家庭协作”
| 功能 | 优先级 |
|------|------|
| 成员管理(Add/Remove) | P1 |
| 成员项目关联手机号 | P1 |
| 多人包裹视图 | P2 |
| 成员取件统计 | P2 |

### v2.2 - “增长引擎”
| 功能 | 优先级 |
|------|------|
| CSV/Excel数据导入 | P2 |
| 取件提醒细分(微信/短信/通知) | P2 |
| 与快递公司合作推广位 | P2 |
| 与驿站合作优先取件 | P2 |

### v3.0 - “平台扩展”
| 功能 | 优先级 |
|------|------|
| iOS版本 | P2 |
| 多语言(中/英) | P2 |
| Android Widget重写 | P2 |

## 3. 优先级矩阵

| 优先级 | 定义 | 示例 |
|------|------|------|
| P0 | 核心体验, 必须发布 | 代取弹窗, 分享格式 |
| P1 | 重要体验, 下个版本 | 成员管理, 深色模式 |
| P2 | 增长/扩展, 按资源排期 | iOS, 合作, 多语言 |

## 4. 时间线
- v2.0: 2026年6月底发布
- v2.1: 2026年8月
- v2.2: 2026年10月
- v3.0: 2027年Q1

## 5. 风险
| 风险 | 影响 | 缓解 |
|------|------|------|
| Android各厂商杀后台 | 通知监控失效 | 厂商特殊指南, 前台服务 |
| iOS SMS权限不开放 | 无法开发iOS版 | 关注 Apple 政策变化 |
| 代取分享滥用 | 垃圾包裹 | 黑名单, 举报机制 |
