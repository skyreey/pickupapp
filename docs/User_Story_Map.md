# 取件通 用户故事地图

## 用户活动链

`
发现包裹 → 查看状态 → 取件 → 管理历史
  |            |            |           |
  |            |            |           → 查看取件统计
  |            |            |           → 导出数据
  |            |            |
  |            |            → 自己取
  |            |            → 找人代取
  |            |
  |            → 看取件码
  |            → 导航到站点
  |            → 拨打电话
  |            → 复制取件码
  |
  → 短信扫描自动发现
  → 通知栏监控发现
  → 微信代取发现
  → OCR截图发现
  → 手动录入
`

## 核心故事列表

### 发现阶段
- US-001: 作为用户, 我希望打开App就看到所有包裹
- US-002: 作为用户, 我希望收到新快递短信时自动识别
- US-003: 作为用户, 我希望通知栏的购物通知被自动捕获
- US-004: 作为用户, 我希望收到家人代取微信后自动弹窗

### 查看阶段
- US-101: 作为用户, 我希望看到取件码大号展示
- US-102: 作为用户, 我希望一键复制取件码
- US-103: 作为用户, 我希望一键导航到取件点
- US-104: 作为用户, 我希望按状态筛选包裹

### 取件阶段
- US-201: 作为用户, 我希望一键标记已取件
- US-202: 作为用户, 我希望将包裹分享给家人代取
- US-203: 作为代取人, 我希望接受后自动导入包裹列表

### 管理阶段
- US-301: 作为用户, 我希望看到取件统计
- US-302: 作为用户, 我希望导出包裹数据
- US-303: 作为用户, 我希望设置取件提醒天数
